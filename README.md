# AI_MED_APP
Ai medical image diagnosis app

MedVision is an end-to-end AI-based healthcare solution that analyzes medical images using deep learning and generates automated diagnostic reports using NLP models. The system integrates computer vision and large language models to assist in faster and more accurate medical decision-making.

Python
TensorFlow / PyTorch
CNN (Convolutional Neural Networks)
NLP (LLM – Gemini / GPT-based models)
Flask / FastAPI
React (Frontend)
MongoDB / Firebase

🔹 Features
🧠 Image Classification – Detects diseases from medical scans (X-ray, MRI, etc.)
📊 Automated Report Generation – Converts predictions into readable medical reports
📱 User-Friendly Interface – Mobile/web app for easy upload & results
⚡ Real-time Processing – Fast prediction and report output
🔗 End-to-End Pipeline – Input → Preprocessing → CNN → LLM → Output
